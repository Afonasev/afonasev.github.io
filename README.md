# afonasev.github.io
![Blog auto-build](https://github.com/afonasev/afonasev.github.io/workflows/Blog%20auto-build/badge.svg)

All hugo commands must be runned in `/src` directory.

## Local serving
`hugo serve`

## Build static files
`hugo -D`

## Create post
`hugo new posts/my-post.md`
